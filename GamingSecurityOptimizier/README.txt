                        NEURVANCE - AIGAMING SUITE

INSTALLATION & USAGE
--------------------

1. Extract all files to a folder on your computer

2. Double-click AIGaming.bat to launch the application
   (or double-click AIGaming.exe directly)

3. That's it! No installation needed.

================================================================================

REQUIREMENTS
------------

• Windows 10 or newer
• Python 3.10+ with PyTorch and Transformers (for AI features)
• ~13 MB disk space

================================================================================

WHAT'S INCLUDED
---------------

• AIGaming.bat        - Launcher script (run this to start)
• AIGaming.exe        - Standalone application (13 MB)
• Neurvance.ico       - Application icon
• README.txt          - This file

Developer: Adam Samer Daabas

================================================================================

TROUBLESHOOTING
---------------

Q: Nothing happens when I click AIGaming.bat
A: Make sure both AIGaming.bat and AIGaming.exe are in the same folder

Q: AI features aren't working
A: Python with torch/transformers must be installed on your system
   Model path: D:\Models\Nanbeige4-3B-Thinking-2511

Q: Can I move the files?
A: Yes, keep all 3 files (bat, exe, ico) together in the same folder
